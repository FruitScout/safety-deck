Etapas 2 y 3 · Bloqueo de Seguridad Química y Monitoreo Drone
Cada Aspersión Bloqueada. Cada Turno Monitoreado.

Bloqueo de Seguridad en Aplicación Química

  • El aplicador escanea el código de barras del químico antes de mezclar. OCR de etiqueta consulta la base de datos COFEPRIS para los requisitos específicos de EPP.
  • El trabajador toma un selfie. La compuerta CV verifica contra la EPP específica que ese químico requiere — una mascarilla de tela vs. respirador de doble cartucho se detecta siempre.
  • El sistema bloquea físicamente el temporizador de "Iniciar Aplicación" y alerta al agrónomo si falla la verificación.

Monitoreo Autónomo Durante el Turno

  • Mientras el drone realiza escaneos rutinarios, ejecuta automáticamente el modelo EPP sobre cualquier trabajador en el encuadre.
  • Mapa de calor de violaciones con GPS — "Trabajador sin Protección Ocular en Surco 47, Sector B3."
  • Puntuación Semanal de Seguridad por cuadrilla — vinculada a bonos de producción, convirtiendo el cumplimiento en un incentivo.
